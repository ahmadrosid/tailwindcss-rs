# Lightweight Tailwindcss

```shell
tailwind-rs 0.1.0
Lightweight tailwindcss!

USAGE:
    tailwind-rs --source <SOURCE> --output <OUTPUT>

OPTIONS:
    -h, --help               Print help information
    -o, --output <OUTPUT>    Css output path
    -s, --source <SOURCE>    Source directories for html files!
    -V, --version            Print version information
```
